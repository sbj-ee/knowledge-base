# Knowledge Base

A searchable reference site aggregating selected public references from
[github.com/sbj-ee](https://github.com/sbj-ee). Built with
[MkDocs Material](https://squidfunk.github.io/mkdocs-material/).

## Sections

<div class="grid cards" markdown>

-   :material-flash: **[Electrical Engineering Reference](ee/index.md)**

    Interactive marimo notebooks accompanying the *Electrical Engineering Reference*
    book — power, communications, semiconductors, control systems, signal processing,
    electromagnetics, and more.

-   :material-console: **[tmux Cheat Sheet](tmux/index.md)**

    A comprehensive reference for tmux keyboard shortcuts and commands — sessions,
    windows, panes, copy mode, and more.

</div>
