# Knowledge Base

A searchable reference site aggregating selected public references from
[github.com/sbj-ee](https://github.com/sbj-ee). Built with
[MkDocs Material](https://squidfunk.github.io/mkdocs-material/).

## Reference

<div class="grid cards" markdown>

-   :material-flash: **[Electrical Engineering Reference](ee/index.md)**

    Interactive marimo notebooks accompanying the *Electrical Engineering Reference*
    book — power, communications, semiconductors, control systems, signal processing,
    electromagnetics, and more.

-   :material-language-c: **[C Programming](c/index.md)**

    A structured C programming learning project covering fundamentals through
    systems programming — types, pointers, memory, concurrency, processes,
    sockets, and more, with progressive exercises and per-topic reference sheets.

-   :material-console: **[tmux Cheat Sheet](tmux/index.md)**

    A comprehensive reference for tmux keyboard shortcuts and commands — sessions,
    windows, panes, copy mode, and more.

</div>

## Network Automation

<div class="grid cards" markdown>

-   :material-ansible: **[Ansible Playbooks](ansible/index.md)**

    Reusable playbooks for server management and configuration — package management,
    OS updates, Python build prep, MySQL, and fact gathering.

-   :material-lan: **[NetBox (pynetbox) Examples](netbox/index.md)**

    Python scripts and a reusable `netbox_utils` library for NetBox automation —
    IP/prefix management, BGP sessions, interface operations, and VLANs.

</div>
